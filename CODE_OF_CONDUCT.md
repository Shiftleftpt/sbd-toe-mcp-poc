# Código de Conduta

Este projeto adota o [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Compromisso

Queremos um ambiente técnico, respeitador e útil para contribuidores, utilizadores e mantenedores.

Não são tolerados:

- assédio, intimidação ou ataques pessoais
- linguagem discriminatória
- comportamento abusivo em issues, PRs, reviews ou discussões

## Âmbito

Aplica-se a:

- issues e pull requests
- revisões de código
- GitHub Discussions
- comunicações oficiais relacionadas com este repositório

## Reporte

Comportamentos inadequados podem ser reportados em privado para:

- `security@shiftleft.pt`

Os reportes serão tratados de forma confidencial.
